    • Hypixel Location Tracker

    Ce logiciel permet de suivre la position d'un joueur jouant sur le Skyblock d'Hypixel. Entrez juste le pseudo du joueur et le logiciel s'occupe de tout.
    Ce site utilise les APIs de Mojang pour passer le pseudo à UUID, Crafatar pour récupere la tête du skin et enfin Hypixel pour récupérer la position du joueur.

    Σ Client

    Ce logiciel utilise les scripts Microsoft Visual Basic Scripting et permettent de définir des MessageBoxs qui se lancent a différents moments. Il propose aussi de créer un fichier .txt qui note les caractéristiques de la machine.
